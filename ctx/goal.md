# CatTalk2D (2D Cat Game, Mobile-first)

## One-liner
클릭하면 반응하고, 이동하고, 상태가 변하는 2D 고양이 게임. (채팅/AI는 옵션)

## Scope (MVP - 게임 코어 우선)
1. **2D 고양이 상호작용** (핵심)
   - 화면 클릭 시 고양이 이동
   - 고양이 클릭 시 반응 (애니메이션/이펙트/사운드)
   - 상태 시스템 (기분/친밀도/배고픔 등)

2. **저장/로드 시스템**
   - 고양이 상태 로컬 저장 (JSON)
   - 앱 재실행 시 상태 복원

3. **기본 게임 루프**
   - 시간 경과에 따른 상태 변화
   - 상태에 따른 고양이 행동 변화
   - 간단한 UI (상태 표시, 메뉴)

## Optional Features (후순위)
- **채팅 UI**: 고양이에게 말 걸기 (텍스트 입력/로그)
- **Claude API 연동**: 고양이가 LLM으로 대답
- **페르소나 시스템**: 말투/성격 반영
- **대화 기록 저장**: 채팅 내역 관리

## Non-goals (지금 안 함)
- 복잡한 행동 AI (단순 상태 기반만)
- 고급 조명/효과/URP 전환
- 온디바이스 LLM
- 멀티플레이어

## Platform
- Windows 개발/테스트 우선
- Android 빌드 목표
- 추후 iOS 고려

## Architecture Philosophy
**게임 먼저, 채팅은 나중**
- Day 1~3: 게임 코어 (이동/상태/저장) 완성
- Day 4~5: 채팅/Claude 연동 (선택적)
- Day 6~7: 폴리싱/모바일 입력

이렇게 하면 "클릭하면 반응하는 고양이 게임"이 먼저 완성되고,
채팅 기능은 그 위에 얹는 구조가 됨.
